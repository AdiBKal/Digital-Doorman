# Digital-Doorman
Digital Doorman : Hands-Free Door Entry &amp; Reception System

### Digital Doorman is a smart, hands-free door access and monitoring system built using the ESP32 microcontroller. It combines presence detection, touch-based interaction, wireless control, and real-time alerts to create a simple yet effective smart security solution.


## Features


Ultrasonic sensor detects nearby presence and activates status LED

Touch sensor works as a doorbell and triggers a buzzer

Servo motor controls door lock/unlock mechanism

Wireless door control via Bluetooth using a mobile device

Real-time activity notifications sent to the user

Simple command-based Bluetooth interface

Low-cost, scalable, and suitable for automation applications


## Applications


Smart home door security

Hotel room access control

Restricted area monitoring (labs, server rooms)

Elderly & disability care entry systems

Smart gates and parking access

Multi-level parking systems
